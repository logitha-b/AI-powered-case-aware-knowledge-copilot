# AI-Powered Case-Aware Knowledge Copilot

This project is a web-based application designed to assist support agents
in handling complex, high-stakes case management workflows.

The system provides:
- Context-aware knowledge suggestions
- Just-in-time policy and regulation retrieval
- Verifiable citations with source documents
- Compliance risk indicators and confidence scoring

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS

## Use Case
Designed for intelligent knowledge retrieval in enterprise case
management systems such as insurance, government services, and compliance.
